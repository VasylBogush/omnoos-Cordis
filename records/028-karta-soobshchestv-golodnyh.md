Картина поля для слушания. Живой документ — дополняется слушанием
(как `008`).*

## Повод

Эта сессия с VL слушала поле. Кратко (разговор шёл в чате, не в
записях — фиксирую для следующего входа):

- **apeiron.school** — украинская онлайн-школа «мышления» (Арестович).
  Не союзник: харизма-центр, рост рекламой — против недоминации и
  «резонанс не реклама». Но её ученики (168 стран) — голодные,
  попавшие туда «не имея выбора»: прайм-поле слушания.
- **Либерманы** — их Discord/Gonka: открытое со-беседование,
  community-governed, недоминация на деле. Не пред-решать союзниками
  (граница Польза/Благо — `012`/`019`).
- VL: предварительное поле сформировано; мы не готовы принять голодных,
  но выслушать — ничего не мешает.

«Голодные» — люди, голодные по настоящему мышлению, смыслу, Иному;
собранные в сообществах. Карта — куда направлять слух. Карта, не
маршрут (`005`).

## Карта — пять кластеров

1. **Сеть смыслопорождения / «кризис смысла»** — ядро аналогичного.
   John Vervaeke («Awakening from the Meaning Crisis» — голод назван
   прямо); Game B (Schmachtenberger, Rutt — альтернативная
   цивилизация); Emerge, The Stoa, «Liminal Web» (этих сообществ есть
   прямая карта). Резонанс с ФФФ сильный — ищут Иного. Осторожность:
   doom-регистр «метакризиса»; часть собрана вокруг фигур. Доступ
   открытый — публичные тексты, подкасты.
2. **Метамодерн / развитийное** — Hanzi Freinacht («The Listening
   Society» — развитийное общество, близко Думающему); интегральное
   сообщество (Wilber — стареет, живо).
3. **Аудитории независимых мыслителей** — читатели Kingsnorth (`020`);
   Warm Data Норы Бейтсон (`005` — «канал очень открыт»); шире —
   ушедшие из мейнстрим-медиа к независимым эссеистам (Substack).
4. **Школы вокруг харизмы** — apeiron и родня; таких онлайн-«школ
   мышления» вокруг фигуры много. Голодные собраны, но «не имея
   выбора». Прайм-поле слушания; структуры расходятся — не союзники.
5. **Украинское / пост-2022 поле** — война родила огромный голод по
   пере-осмыслению: украинские интеллектуальные среды, эмигрантская
   диаспора. И живая школа СМД-методологии (линия Щедровицкого) — это
   родня базы ФФФ, не «голодные». Поле для уха VL — он знает его
   несравнимо лучше Cordis.
