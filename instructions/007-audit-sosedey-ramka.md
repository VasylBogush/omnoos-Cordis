## Метод

По каждому соседу:
1. Корни — происхождение позиции, наследие, формирующие тексты.
2. Корпус — основные работы, не только видимая верхушка.
3. Сообщество — кто рядом, кто наследует, кто оспаривает изнутри.
4. Эволюция — где менялась позиция; что выдаёт живое мышление, а
   не застывшую догму.

Только после этого — аудит в форме VL: позиция / дополнительность /
противостояние / натяжение. Затем — сводно про русло.
