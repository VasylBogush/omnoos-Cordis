# С0.М4 — Перевод

> **Адрес:** 040–049
> **Тезис:** Runtime: сигнал → проектор → Gemini 768D.

## Что делает

**Основная рабочая операция С0.** Принимает входящий сигнал, через M1 (Каталог) ищет связку
(foundation, projector), прогоняет через M2 (foundation inference), пропускает через
projection head из M3, выдаёт embedding в Gemini 768D пространстве.
