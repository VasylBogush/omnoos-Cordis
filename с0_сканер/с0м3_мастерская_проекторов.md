# С0.М3 — Мастерская проекторов

> **Адрес:** 030–039
> **Тезис:** Где живут projection heads — обучение, валидация, хранение весов, обновление.

## Что делает

Модуль про **жизненный цикл проекторов**: обучение MLP-голов, которые переводят native-latent
foundation модели в Gemini 768D через contrastive learning на anchor-парах. Хранение весов,
валидация на отложенных парах, версионирование, обновление при появлении новых anchors.
